Using A Bootstrap Template

Bootstrap is a framework created by Twitter. It helps us style our application really nicely for different screens using a grid layout system. Instead of using CSS to create hacky code that allows two divs to sit next to each other, we can use bootstrap classes that will provide the same functionality out of the box. We can also add classes that will style our divs nicely when the view changes to tablet or mobile. Talk about boss.

JavaScript and JQuery for Back-End Needs

Let's say you need to send an email out from your website. How do you think you're going to do that? To do so, you'll need to utilize a scripting language such as PHP, JavaScript, or Node.JS. Because you're in my world right now, I'm going to show you guys how to utilize JavaScript. My rationale behind this is that it's a heavily, heavily utilized language whose functionality is always increasing. Learning and using JavaScript makes us future-proof, flexible, and very valuable within a team.
